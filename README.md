# CodeAlpha Project Management Tool

## Overview
A full-stack Project Management Tool developed as part of the CodeAlpha Full Stack Development Internship.

## Features
- Create and manage projects
- Assign tasks
- Track project progress
- User-friendly interface
- Backend support using Node.js and Express.js

## Technologies Used
- HTML
- CSS
- JavaScript
- Node.js
- Express.js

## Project Structure
- index.html
- style.css
- script.js
- server.js
- package.json

## How to Run

1. Install dependencies:
   npm install

2. Start the server:
   npm start

3. Open browser:
   http://localhost:3000
